# ストーリー骨格

## 状態

初期検討中。ここに記載された内容は、ゲームシステムと縦切り開発の進行に合わせて更新します。

## 現在確定している前提

- 主人公は比較的弱い存在から始まる。
- 主人公は「魔王になる」ことを目指す。
- 「魔王になる」は単純な悪役化に限定しない。
- 力、立場、責任、選択を物語とゲームシステムの両方で扱える余地を残す。

## 今後定義する項目

### 主人公

- 名前
- 出自
- 現在の立場
- 魔王を目指す理由
- 初期能力

### 世界

- 魔王という存在の意味
- 現在の支配構造
- 人間・魔族・その他勢力の関係
- 主人公にとっての主要な対立

### プレイヤーの選択

- 選択が物語へ与える影響
- 成長や進化との関係
- NPCとの関係変化

## 次の縦切りで必要な最小ストーリー

```text
NPCと会う
  -> 小さな目的を受け取る
  -> 探索または戦闘を行う
  -> 結果を報告する
  -> 報酬または進行変化を得る
```

この最小ループを成立させる内容から先に具体化します。
