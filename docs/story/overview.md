# ストーリー骨格

- Status: Draft

## 現在確定している前提

- 主人公は比較的弱い存在から始まる。
- 主人公は「魔王になる」ことを目指す。
- 「魔王になる」は単純な悪役化に限定しない。
- 力、立場、責任、選択を物語とゲームシステムの両方で扱える余地を残す。

## 今後定義する項目

- 主人公の出自と立場
- 魔王を目指す理由
- 魔王という存在の意味
- 主要勢力と対立
- プレイヤー選択の影響
- 成長・進化と物語の関係

## 次の縦切りで必要な最小ストーリー

```text
NPCと会う
  -> 目的を受け取る
  -> 探索または戦闘を行う
  -> 成長・報酬を得る
  -> 結果を報告する
  -> 進行変化を得る
```
