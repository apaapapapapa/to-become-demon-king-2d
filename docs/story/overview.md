# ストーリー骨格

- Status: Draft

ゲーム全体の中核コンセプトは [ゲームビジョン](../game/vision.md) を正とします。この文書では、物語制作のために具体化した主人公の経歴、中心対立、未確定事項を管理します。

## 主人公の出自

主人公は森の中で魔物として生まれます。生まれた時点では強大な存在ではなく、自身と同じく弱い一体の魔物に育てられます。

その育て親となった魔物は後に殺され、主人公は庇護と居場所を失います。主人公が人間社会へ紛れ込む最初の理由は、世界を支配するためではなく、生き延びるため、そして誰かと繋がる場所を得るためです。

## 人間社会との関係

主人公にとって、人間は当初「魔物を殺す、信用できない存在」です。それでも人間社会で正体を隠しながら冒険者として活動するうちに、善意を向ける人、共に戦う人、信頼できる人にも出会います。

物語前半では、人間への警戒心を残しながらも「人間すべてが同じではない」と主人公が学んでいく過程を描きます。

## 裏切りと覚醒

主人公が冒険者として経験を積み、魔物としても大きな力を身につけ始めた段階で、人間側からの裏切りを経験します。

この裏切りは単なる挫折ではなく、主人公の内側に眠っていた、あるいは成長によって到達可能になった大きな力の覚醒を促す転換点とします。

## 中心対立

覚醒後の主人公には、少なくとも次の二つの方向が生まれます。

- 自分を裏切った人間社会を見限り、その力を受け入れて「魔王」となる。
- 裏切りを受けても、それまでに出会った善良な人々との繋がりを根拠に、人間を信じる可能性を捨てない。

「魔王」は最初から主人公が目指していた肩書ではなく、喪失、力、裏切り、信頼の積み重ねの先で現れる選択肢・立場として扱います。

最終的にどの程度プレイヤー選択で分岐させるか、単一の結末に収束させるかは未確定です。

## 今後定義する項目

- 主人公の種族、名前、外見、能力の起源
- 育て親の魔物の人物像と、殺害に至る具体的な事件
- 主人公が人間社会へ自然に紛れ込める仕組み
- 冒険者として築く主要人物との関係
- 主人公を裏切る人物・組織、その動機と事件の詳細
- 覚醒する力とEvolutionシステムの関係
- 「魔王」という存在の世界内での歴史的・政治的な意味
- 主要勢力が主人公の正体と覚醒へどう反応するか
- プレイヤー選択とエンディング分岐の範囲
