# 仕様書

このセクションは、実装と同期して管理する機能仕様を置きます。

現在の仕様:

- [入力仕様](./input.md)
- [戦闘仕様](./combat.md)
- [インタラクション仕様](./interaction.md)
- [成長仕様](./progression.md)
- [セーブ仕様](./save.md)

今後、会話、クエスト、Skill、Evolution、Scene遷移などを追加するときは、機能ごとに仕様書を追加します。

## 仕様書に書くこと

- ユーザー／ゲーム上の振る舞い
- 状態遷移
- 成功条件・失敗条件
- 他機能との境界
- Runtime値のSource of Truthへの参照
- Stable Content ID
- 未実装事項

クラス内部の詳細な実装手順は `design/technical-design.md` に寄せます。
