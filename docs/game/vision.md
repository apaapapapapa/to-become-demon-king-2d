# ゲームビジョン

## 中核コンセプト

『To Become Demon King』は、アイソメトリック2D／2.5DのピクセルアートRPGです。プレイヤーは比較的弱い存在から始まり、探索、交流、戦闘、成長を重ねながら「魔王になる」ことを目指します。

「魔王になる」は単純な悪役化に限定せず、力、立場、責任、選択を物語とゲームシステムの両方で扱える余地を残します。

## 参考作品との距離

Witchbrookなどはアイソメトリック視点やピクセルアート表現の参考にしますが、直接的な模倣や同等規模の生活シミュレーションを目標にはしません。

## ゲームプレイの柱

1. 探索
2. キャラクターとのインタラクション
3. 戦闘
4. 成長／進行

新機能を追加するときは、どの柱を強化するのかを明確にします。

## 現在のPrototype

```text
フィールドを移動する
  -> NPCとInteractする
  -> 敵をAttackする
  -> 撃破する
  -> RewardService経由で経験値を得る
  -> CharacterProgressionStateの経験値・レベルが更新される
  -> Dodgeで回避する
  -> PauseでGameplay/UI入力を切り替える
```

## 次の縦切り目標

```text
NPCと会話する
  -> 目的を受け取る
  -> 探索または戦闘を行う
  -> 成長・報酬を得る
  -> 結果を報告する
  -> 世界や進行に変化が起きる
```

## 物語方針

最初から全シナリオを固定せず、主人公、魔王を目指す理由、世界の雰囲気、中心対立、プレイヤー選択の意味から段階的に具体化します。

詳細は `story/` で管理します。

## ビジュアル方針

- アイソメトリック、またはそれに近い固定視点
- Isometric Tilemapを基礎にした2Dワールド
- Yソートと前景による奥行き表現
- 主要アートはプロジェクト管理アセットを正とする
- URP 2D Lightingは雰囲気と視認性の補強に利用する

## 対象プラットフォーム

第一ターゲットはWindows / Steamです。将来のコンソール移植を妨げないよう、GameplayをSteam固有APIやマウス専用操作へ直接依存させません。
