# ゲームビジョン

## 中核コンセプト

『To Become Demon King』は、アイソメトリック2D／2.5DのピクセルアートRPGです。プレイヤーは森で生まれた弱い魔物として始まり、生き延びることと誰かとの繋がりを求めて人間社会へ紛れ込みます。探索、交流、戦闘、成長を重ねる中で、人間への不信と信頼の両方を経験していきます。

主人公が力を得た後、人間からの裏切りをきっかけに大きな力が覚醒します。その力をもって人間を見限り「魔王」になるのか、それでも人を信じようとするのかを、物語の中心対立として扱います。

「魔王になる」は最初から決められた悪役化や単純な到達目標ではなく、力、立場、責任、喪失、他者との繋がりに対する主人公の選択として描きます。

## 参考作品との距離

Witchbrook等はアイソメトリック視点やピクセルアート表現の参考にしますが、直接的な模倣や同等規模の生活シミュレーションを目標にはしません。

## ゲームプレイの柱

1. 探索
2. キャラクターとのインタラクション
3. 戦闘
4. 成長／進行

新機能を追加するときは、どの柱を強化するのかを明確にします。

## 物語方針

主人公の出自、人間社会へ入る理由、人間への不信と信頼、裏切りによる覚醒、「魔王になる／ならない」という中心対立を物語の軸とします。

個々の事件、主要人物、勢力との関係、プレイヤー選択をどの範囲までゲームシステムへ反映するかは段階的に具体化します。物語固有の制作情報は [ストーリー](../story/) で管理します。

## ビジュアル方針

- アイソメトリック、またはそれに近い固定視点
- Isometric Tilemapを基礎にした2Dワールド
- Yソートと前景による奥行き表現
- 主要アートはプロジェクト管理アセットを正とする
- URP 2D Lightingは雰囲気と視認性の補強に利用する

## 対象プラットフォーム

第一ターゲットはWindows / Steamです。将来のコンソール移植を妨げない構造を維持します。技術的なPlatform境界は [アーキテクチャ](../design/architecture.md#platform境界) を参照してください。
